Upload the Excel site plan and the branded template — the server writes AI-researched copy, real landmarks, and a map screenshot into one filled slide per site.

ℹ️ Expects one header row with State, Market, Location, Contacts Per Day, Media, Type, Units, W, H, SOV %, Lead Time, Language, Screen Resolution, Screen Loop columns (blank/subtotal rows are skipped automatically). No vendor photos are used — each site's photo placeholder is left blank and the map placeholder gets a geocoded Google Maps screenshot instead. This means the build can take several minutes for large plans (one AI call plus one map lookup plus one landmark lookup per site).
